# Full Stack Projects Collection

A collection of full-stack web applications built using Java, Spring Boot, React, and MySQL.

## 🚀 Projects

### 📝 Blog App
A modern blogging platform that allows users to create, manage, and explore blog posts.

**Features**
- User Authentication
- Create, Edit & Delete Posts
- Category Management
- Responsive UI
- REST API Integration

**Tech Stack**
- React
- Spring Boot
- MySQL
- JWT Authentication

---

### 🤖 HireTrack AI
An AI-powered Applicant Tracking System (ATS) designed to help recruiters manage candidates and analyze resumes using the Gemini API.

*Features 
- Candidate Management
- Resume Upload
- AI Resume Analysis
- Email Integration
- JWT Authentication

*Tech Stack 
- React
- Spring Boot
- MySQL
- REST APIs
- Gemini API

---
